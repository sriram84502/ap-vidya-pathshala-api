Authentication Endpoints

Student Login

POST /.netlify/functions/auth/student

{
  "username": "student001",
  "password": "yourpassword"
}

Teacher Login

POST /.netlify/functions/auth/teacher

{
  "username": "teacher001",
  "password": "yourpassword"
}

Headmaster Login

POST /.netlify/functions/auth/headmaster

{
  "username": "hm001",
  "password": "yourpassword"
}

Admin Login

POST /.netlify/functions/auth/admin

{
  "username": "admin001",
  "password": "yourpassword"
}

Response Format

{
  "message": "User logged in successfully",
  "token": "jwt_token_here"
}
